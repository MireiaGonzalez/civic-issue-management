# Urbanito

Urbanito is a civic issue-management application that facilitates communication between citizens and local decision-makers.

Citizens can report problems in their area by easily creating a ticket with descriptions, categories, locations, and image attachments.

Administrators can review and manage incoming tickets using Urbanito's prioritization and status-update system. They can add internal notes, track progress, and manage the resolution process through an intuitive and simple workflow.

The goal of this application is simplifying and organising issue reporting for citizens and issue resolution by administrators.

This project is built as a full-stack portfolio application using **Next.js**, **React**, **TypeScript**, **Java**, **Spring Boot**, **PostgreSQL**, **Docker**, **testing**, and **clean development practises**.
